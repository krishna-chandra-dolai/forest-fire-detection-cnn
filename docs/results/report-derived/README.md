## Image preparation

The PDF's original embedded figures were used, rather than screenshots of whole report pages. Only deterministic presentation changes were applied: near-black outer-border trimming, a small white margin, 2x Lanczos resizing, and mild sharpening. Photo-heavy figures use high-quality 4:4:4 JPEG; text-heavy figures use PNG.

No generative image reconstruction or metric rewriting was used. These remain historical report-derived results, not a fresh model run.
